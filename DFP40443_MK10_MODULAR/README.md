## Arahan Tugasan

1. **Fork** repositori asal ke akaun GitHub anda.
2. **Clone** repositori yang anda fork ke komputer sendiri.
3. **Jalankan projek PHP tersebut** sehingga ia berfungsi sepenuhnya.
4. **Ubah struktur projek supaya menjadi lebih modular.**
   - Pecahkan bahagian yang wajar.
   - Contoh: Asingkan *header*, *footer*, *navigasi*, dan *data produk* ke fail berasingan.
   - Pastikan tiada kod HTML/CSS yang berulang.
   - Kekalkan fungsi asal.
5. **Push** perubahan ke repositori fork anda.
6. **Buat Pull Request** ke repositori asal.
7. Sediakan nota ringkas dalam *Pull Request*:
   - Apakah yang anda modularkan?
   - Mengapa struktur itu lebih baik?
